# IT Help Desk Database

## Overview
This project implements a relational database to manage IT help desk users, tickets, and technicians. 

## Features
- Relational database design with primary and foreign keys
- SQL queries using JOIN, LEFT JOIN, GROUP BY, HAVING
- Safe updates and deletions with SQL_SAFE_UPDATES

## Technologies
- MySQL
- SQL

## Usage
1. Open `SQL-scripts.sql` in MySQL Workbench or preferred SQL editor.
2. Run the scripts to create tables and insert sample data.
3. Execute queries to analyze ticket volume and technician workload.
